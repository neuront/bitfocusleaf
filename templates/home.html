Project Leaf 是一个非应用性质的站点, 站点上包含一些演示用小程序.

通过直接保存 Project Leaf 的任何页面可以获取其 HTML 源代码. 任何 Project Leaf 页面中的任何源代码 (包括但不限于 HTML, Javascript, CSS) 文件均以下面协议发布

  • Project Leaf 只能保证源代码可能在特定环境下正常使用, 使用源代码产生的任何结果与 Project Leaf 无关.
  • 无论处于何种目的, 任何人或组织都可以获取, 修改, 演绎, 包装, 重发布源代码.

Project Leaf 不提供 API, 亦不保证站点小程序甚至整个站点的稳定性.
